# example-by-clojure

This repository for learning clojure. And it's based on [kim's repository](http://kimh.github.io/clojure-by-example/#about).

## set up clojure

```
$ brew install clojure
$ brew upgrade clojure
```

## Hello, world!

### print Hello World!

```
$ clojure
Clojure 1.10.0
user=> (println "Hello World!")
Hello World!
nil
```

### how to exit REPL

`user=> (System/exit 0)`
or
`ctrl+d`

### help

```
$ clojure --help
```
